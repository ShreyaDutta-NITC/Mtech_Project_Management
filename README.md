# dbms_project
This project consists of admin, faculty and students to work on the project. Each of them can register themselves and login to their own dashboard. Then admin can approve faculty or student. The Faculty can see the completed or ongoing projects on their dashboard, can give marks and grades to the students. The students can upload all the related files of the projects and there will be multiple assessments, the deadline will be set by the project guide. Before the deadline, students have to upload their projects and evaluators will give marks, after the final assessment students can download their score card.

Backend - phpmyadmin
Frontend - html, css, js
